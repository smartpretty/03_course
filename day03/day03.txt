1.使用ssh方式关联远程仓库
https在每次推送时，都需要输入用户名与密码，比较麻烦
配置ssh:
查看本机是否要ssh key
ls -a ~/.ssh
创建生成密钥: 
打开公钥，赋值里面的内容：
admin--.ssh--.pub
远程仓库：
settings--SSH and GPG keys--new SSH key
关联并推送

二、从远程仓库下载内容
(一)方法一
  1.进入目标文件夹,初始化为本地仓库
  2.下载远程仓库内容
    git pull 远程仓库地址.git master
(二)方法二,使用克隆命令,下载的是完整的远程仓库
    git clone 远程仓库地址.git
